HEROVERSE KIDS — FRONTEND IMPLEMENTATION SPEC
==============================================
Read this entire file before writing any code.

PROJECT
-------
HeroVerse Kids is an AI-powered personalized storybook platform where children (age 4–12) become
the hero of their own illustrated, narrated comic story. Parents upload a photo, pick a theme,
and receive a personalized PDF with audio narration.

Repo: https://github.com/vatharsh/HeroKidsUniverse.git
Work only inside: apps/web/src/


TECH STACK
----------
- Next.js 16 (App Router) — read node_modules/next/dist/docs/ before writing any Next.js code
- TypeScript (strict mode)
- Tailwind CSS v4 — config lives in globals.css @theme block. There is NO tailwind.config.js.
- lucide-react for icons
- clsx + tailwind-merge (cn() utility already exists at src/lib/utils.ts)
- Do NOT install any new packages. Only use what is already installed.


DESIGN SYSTEM (already set up in globals.css — use these exact class names)
----------------------------------------------------------------------------
Colors (use as bg-X or text-X):
  brand         → #7C3AED  (purple — primary CTA color)
  brand-dark    → #5B21B6  (hover state)
  brand-light   → #A78BFA
  brand-50      → #F5F3FF
  gold          → #F59E0B  (magic highlights, stars)
  gold-light    → #FDE68A
  space         → #0B0520  (dark hero backgrounds)
  space-mid     → #1A0850
  space-light   → #2D1070
  cream         → #FDFAFF  (light section backgrounds)
  ink           → #120B2E  (headings)
  ink-mid       → #4B4468  (body text)
  ink-muted     → #9489B4  (captions, placeholders)

Utility classes (already defined in globals.css, use directly):
  text-gradient-brand       → purple gradient text
  text-gradient-gold        → gold gradient text
  text-gradient-magic       → purple-to-gold gradient text
  bg-space-gradient         → dark radial space background
  bg-brand-gradient         → solid purple gradient background
  shadow-brand              → purple glow box-shadow
  shadow-card               → soft card box-shadow
  shadow-gold               → gold glow box-shadow
  animate-float             → gentle up-down float (4s loop)
  animate-float-slow        → slower float (6s loop)
  animate-sparkle           → scale + opacity pulse
  animate-fade-in-up        → entrance slide-up animation
  animate-twinkle           → opacity + scale pulse
  animate-shimmer           → horizontal shimmer sweep

Font classes (Tailwind v4 syntax for CSS variable fonts):
  Headings (Nunito): font-[family-name:var(--font-display)]
  Body (Inter):      font-[family-name:var(--font-body)]


BRAND VOICE
-----------
- Warm, magical, premium — NOT cartoonish
- Parents are the buyer, child is the star
- Copy speaks to emotion: "magical memory", "become a hero"
- Light cream bg for content sections, dark space bg for hero/accent sections
- No dark mode needed

FILE 1: src/components/shared/StarField.tsx
─────────────────────────────────────────────────────────────────────────────
Decorative background component — 60 twinkling white star dots.

- Outer div: position absolute, inset-0, pointer-events-none, overflow-hidden
- Pre-define an array of 60 star objects (do NOT use Math.random() at render time
  — hardcode the values to avoid hydration mismatch):
  Each object: { x: number, y: number, size: number, duration: number, delay: number }
  x: 0–100 (percentage), y: 0–100, size: 1–3 (px), duration: 1.5–4 (s), delay: 0–3 (s)
- Render each as a <div> with:
    position absolute, left: x%, top: y%, width/height: size px
    borderRadius 50%, backgroundColor white
    animation: twinkle {duration}s ease-in-out infinite {delay}s (inline style)
